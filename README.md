# Quick-Folder

一个使用 PyQt6 开发的快捷文件夹管理工具，支持添加、打开、重命名、排序和删除常用文件夹，并将数据本地保存，方便日常快速访问。

![程序截图](images/demo01.png)

## 操作说明

- 在输入框中输入文件夹路径，点击 **添加** 可将其加入列表
- 双击列表中的文件夹可直接打开对应目录
- 按 **F2** 可重命名当前选中的文件夹
- 按 **Delete** 可删除当前选中的文件夹
- 支持通过拖拽调整文件夹在列表中的顺序
- 关闭窗口后程序不会退出，而是最小化到系统托盘
- 点击托盘图标或使用托盘菜单中的 **打开** 可重新显示主窗口
- 使用托盘菜单中的 **退出** 可彻底关闭程序
